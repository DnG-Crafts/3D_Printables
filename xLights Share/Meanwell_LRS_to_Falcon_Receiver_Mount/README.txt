# Summary

This mount attaches a Falcon 4 port receiver to a Meanwell LRS power supply. I did not design the 3 holes for the Falcon receiver with threads, so I recommend using a 6/32 tap to thread the holes and then secure the receiver with three 6/32 nylon machine screws. Then attach the mount to the side of the power supply with two M4 screws.
