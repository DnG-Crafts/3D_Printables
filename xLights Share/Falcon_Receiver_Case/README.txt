# Summary

This is a case for the Falcon Receiver.

There are 2 options for the base, one that will align with the holes on a CG1500 box and a option without the tabs for the CG box.

You will need:
3x M2.5x10mm screws to hold the receiver to the base.
4x M2.5x10mm screws to hold the top to the base.
This is the one I used:
https://www.amazon.com/gp/product/B012TF586O/ref=ox_sc_act_title_1?smid=A1THAZDOWP300U&psc=1
